# מדריך התקנה מפורט - Anti-Spoiler Extension

## שלב 1: יצירת אייקונים

יש לך 3 אפשרויות ליצירת האייקונים:

### אפשרות 1: שימוש בקובץ HTML (הכי פשוט) ⭐

1. פתח את הקובץ `create-icons.html` בדפדפן Chrome או Firefox
2. תראה 3 אייקונים על המסך
3. לחץ על כל אייקון - הוא יוריד אוטומטית
4. ודא שהקבצים נשמרו בשמות:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`
   - בתיקייה `AntiSpoiler`

### אפשרות 2: יצירת אייקונים פשוטים ידנית

אם אתה לא רוצה להשתמש ב-HTML, אתה יכול ליצור 3 קבצי PNG פשוטים:
- `icon16.png` - 16x16 פיקסלים
- `icon48.png` - 48x48 פיקסלים  
- `icon128.png` - 128x128 פיקסלים

אפשר להשתמש בכל עורך תמונות או אפילו ליצור ריבועים צבעוניים פשוטים.

### אפשרות 3: שימוש בסקריפט Python

```bash
# יצירת סביבה וירטואלית
python3 -m venv venv
source venv/bin/activate

# התקנת Pillow
pip install Pillow

# יצירת האייקונים
python create-icons.py
```

## שלב 2: התקנת ההרחבה ב-Chrome

1. **פתח את Chrome**

2. **נווט לדף ההרחבות:**
   - כתוב בשורת הכתובת: `chrome://extensions/`
   - או: לחץ על 3 הנקודות (☰) → Extensions → Manage extensions

3. **הפעל "מצב מפתח" (Developer mode):**
   - בפינה הימנית העליונה תראה מתג "Developer mode"
   - לחץ עליו כדי להפעיל (המתג יהפוך לכחול)

4. **טען את ההרחבה:**
   - לחץ על הכפתור "Load unpacked" (טען הרחבה לא ארוזה)
   - בחר את התיקייה `AntiSpoiler` (התיקייה שמכילה את manifest.json)
   - לחץ "Select" או "Open"

5. **ההרחבה מותקנת! 🎉**
   - תראה את ההרחבה ברשימה
   - תראה את האייקון שלה בסרגל הכלים של Chrome

## שלב 3: שימוש בהרחבה

### הפעלה אוטומטית
ההרחבה פועלת מיד לאחר ההתקנה! הוסף מילות מפתח ברשימה והיא תחסום תוכן שמכיל אותן בכל העמודים שאתה מבקר בהם.

### שליטה בהרחבה
לחץ על אייקון ההרחבה בסרגל הכלים כדי לפתוח את החלון:

- **סטטוס**: תראה אם ההרחבה פעילה או מושבתת
- **רענן עמוד**: לרענן את העמוד הנוכחי
- **השבת/הפעל**: לכבות או להפעיל את ההרחבה

### בדיקה שההרחבה עובדת
1. פתח עמוד חדש ב-Chrome
2. חפש בגוגל מילה שהוספת לרשימה
3. התוכן הקשור אמור להיות מוסתר!

## פתרון בעיות

### ההרחבה לא מופיעה ברשימה
- ודא ש"Developer mode" מופעל
- ודא שבחרת את התיקייה הנכונה (זו שמכילה את manifest.json)

### האייקונים לא מופיעים
- ודא שיצרת את 3 קבצי האייקונים (icon16.png, icon48.png, icon128.png)
- ודא שהקבצים נמצאים באותה תיקייה כמו manifest.json

### ההרחבה לא חוסמת תוכן
- ודא שההרחבה פעילה (לחץ על האייקון ובדוק את הסטטוס)
- נסה לרענן את העמוד (F5 או Ctrl+R)
- בדוק שהתוכן באמת מכיל את מילות המפתח שהוספת

## עדכון ההרחבה

אם עדכנת את הקוד:
1. לך ל-`chrome://extensions/`
2. לחץ על כפתור הרענון (🔄) ליד ההרחבה
3. או הסר והתקן מחדש

---

## הסרת ההרחבה

1. לך ל-`chrome://extensions/`
2. לחץ על "Remove" ליד ההרחבה
3. אשר את ההסרה
