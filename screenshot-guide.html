📸 Screenshot Guide for Chrome Web Store

Chrome Web Store requires at least one screenshot. 3–5 screenshots are recommended. Recommended size: 1280×800 or 640×400.

Screenshots to Capture

1. Extension Popup

Click the extension icon in the toolbar and capture the popup window.

  • Load the extension (Load unpacked)
  • Click the extension icon
  • Add a few sample keywords
  • Take a screenshot (Cmd+Shift+4 on Mac, Win+Shift+S on Windows)

2. Blocked Content on a Website

Capture an example of a website where spoiler content is hidden.

  • Open android-test-page.html or any site with relevant content
  • Add keywords that match the content (e.g. "example", "test-spoiler")
  • Refresh the page – the content should be covered
  • Take a screenshot of the page with blocked content

3. Before/After Comparison (Optional)

If possible, capture the same page with the extension off and on.

💡 Tip: If screenshots are smaller than 1280×800, enlarge the window before capturing or use an image editor to add a frame.

Where to Upload

In the Chrome Web Store upload form, in the "Screenshots" section – drag and drop the files or click "Upload".
