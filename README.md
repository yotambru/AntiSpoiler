# Anti-Spoiler Chrome Extension

הרחבה ל-Google Chrome שחוסמת אוטומטית תוכן ספויילרים בהתאם למילות המפתח שבחרת, בכל העמודים שאתה מבקר בהם.

## תכונות

- ✅ חוסם אוטומטית תוכן בהתאם למילות המפתח שלך
- ✅ עובד על כל העמודים באינטרנט
- ✅ מזהה תוכן דינמי שנטען לאחר מכן
- ✅ ממשק נוח להפעלה/השבתה
- ✅ תמיכה בעברית ואנגלית

## התקנה

### שלב 1: יצירת אייקונים

לפני ההתקנה, יש ליצור את קבצי האייקונים:

**אפשרות 1 - שימוש בקובץ HTML:**
1. פתח את הקובץ `create-icons.html` בדפדפן
2. לחץ על כל אחד מהאייקונים כדי להוריד אותו
3. שמור את הקבצים בשמות: `icon16.png`, `icon48.png`, `icon128.png` בתיקיית הפרויקט

**אפשרות 2 - שימוש בסקריפט Python:**
```bash
pip install Pillow
python create-icons.py
```

**אפשרות 3 - יצירה ידנית:**
צור 3 קבצי PNG בגדלים 16x16, 48x48, 128x128 פיקסלים ושמור אותם בשמות:
- `icon16.png`
- `icon48.png`
- `icon128.png`

### שלב 2: התקנת ההרחבה

1. פתח את Chrome ונווט ל-`chrome://extensions/`
2. הפעל "מצב מפתח" (Developer mode) בפינה הימנית העליונה
3. לחץ על "טען הרחבה לא ארוזה" (Load unpacked)
4. בחר את תיקיית הפרויקט `AntiSpoiler`
5. ההרחבה תותקן ותתחיל לעבוד אוטומטית

## שימוש

לאחר ההתקנה, הוסף מילות מפתח ברשימה וההרחבה תחסום תוכן שמכיל אותן.

לחץ על אייקון ההרחבה בסרגל הכלים כדי:
- לראות את סטטוס ההרחבה
- לרענן את העמוד הנוכחי
- להפעיל או להשבית את ההרחבה

## איך זה עובד

ההרחבה משתמשת ב-Content Script שרץ על כל עמוד ומחפש:
- טקסט המכיל את מילות המפתח שהוספת לרשימה
- תכונות אלמנטים כמו alt, title, aria-label
- תוכן דינמי שנטען לאחר מכן באמצעות MutationObserver

כל אלמנט שמכיל תוכן רלוונטי מוסתר אוטומטית.

## פיתוח

### מבנה הקבצים

- `manifest.json` - קובץ הגדרות ההרחבה
- `content.js` - סקריפט שרץ על כל עמוד ומסנן תוכן
- `background.js` - Service Worker של ההרחבה
- `popup.html/css/js` - ממשק המשתמש של ההרחבה

### הוספת מילות מפתח

כדי להוסיף מילות מפתח נוספות, ערוך את מערך `keywords` בקובץ `content.js`:

```javascript
const keywords = [
  'תוצאות משחק',
  'סוף סדרה',
  // הוסף את המילות מפתח שלך...
];
```
