Privacy Policy

Last updated: March 16, 2025

Summary: Anti-Spoiler does not collect, transmit, or share any personal data. All user settings and keywords are stored exclusively on your own device using the browser's built-in local storage. The extension makes outbound requests only to public knowledge-base APIs (Wikidata and Wikipedia) solely to help you discover related spoiler keywords — no personal information is included in those requests.

1. Who We Are

Anti-Spoiler is a Chrome browser extension developed and maintained as an independent open-source project. References to "we", "us", or "our" in this policy refer to the extension's developer(s).

2. Data We Collect

We do not collect any personal or sensitive user data. No data is ever sent to our servers — because we do not operate any servers.

The extension stores the following information locally on your device only using the Chrome Storage API (chrome.storage.local):

  • Spoiler keywords: The list of keywords and phrases you add to block spoilers.
  • Enabled/disabled state: Whether the extension is currently active.
  • Game/event state: A record of the current game or event (e.g. date, whether you have marked it as watched), used to determine when to display spoiler blocking.
  • Keyword suggestion cache: When you request related keyword suggestions, the results returned by the Wikidata public API are cached locally so future lookups are faster and do not require repeated network requests.

None of this information ever leaves your device. It is not transmitted to us or any third party.

3. Third-Party API Requests

When you use the "Find related keywords" feature, the extension makes outbound HTTPS requests to the following public, read-only APIs:

Wikidata API

  • URL: https://www.wikidata.org
  • Purpose: To search for entities (people, teams, events) related to the keyword you typed, and retrieve their name variants and aliases.
  • Data sent: The keyword text you typed. No personal information is included.
  • Wikidata's privacy policy: https://foundation.wikimedia.org/wiki/Privacy_policy

Wikipedia API

  • URL: https://en.wikipedia.org (and language-specific variants)
  • Purpose: To retrieve publicly available article text (for sports figures) in order to identify the current team or organization associated with a keyword.
  • Data sent: The entity name or article title derived from the Wikidata search. No personal information is included.
  • Wikipedia's privacy policy: https://foundation.wikimedia.org/wiki/Privacy_policy

These API calls are made directly from your browser. The extension developer does not receive, intercept, or log any of these requests or their responses.

4. Data Handling and Storage

  • Storage location: All data is stored locally on your device using chrome.storage.local, which is sandboxed to this extension.
  • Retention: Data persists until you uninstall the extension or manually clear it. There is no automatic expiry, except for the keyword suggestion cache which is automatically refreshed after 7 days.
  • Access: Only this extension can access its own storage. We cannot access, view, or retrieve your stored data remotely.
  • Encryption: Chrome's local storage is protected by your operating system's user account security and Chrome's sandboxing model. We do not apply additional encryption.

5. Data Sharing

We do not share any data with any third party. Because we do not collect or receive any data from you, there is nothing to share.

The only outbound network communication the extension performs is the public API requests to Wikidata and Wikipedia described in Section 3 above, which are initiated solely on your behalf and contain only the keyword text you typed.

6. Permissions Justification

The extension requests the following browser permissions:

  • storage: Required to save your keywords and preferences locally on your device.
  • activeTab: Required to interact with the currently open tab when you use the extension popup.
  • tabs / windows: Required to apply spoiler blocking consistently across browser windows and tab navigation events.
  • host_permissions (<all_urls>): Required to inject the content script that scans and hides spoiler content on every webpage you visit. No browsing data is collected or transmitted as a result of this permission.

7. Children's Privacy

This extension does not knowingly collect data from anyone, including children under 13 (or the applicable age of digital consent in your jurisdiction). Since no data is collected at all, this extension is safe to use regardless of age.

8. Your Rights and Data Deletion

Because all data is stored locally on your device, you have full control over it at all times:

  • You can clear all stored data by uninstalling the extension.
  • You can remove individual keywords at any time from the extension popup.
  • You can clear Chrome extension storage through Chrome's developer tools (chrome://extensions → Details → Clear data).

Since we never receive or store any of your data on our end, there is no additional action needed from us to delete your data.

9. Changes to This Policy

If we update this privacy policy, we will revise the "Last updated" date at the top of this page and publish the updated version at the same URL. We encourage you to review this policy periodically. Continued use of the extension after a policy update constitutes acceptance of the updated terms.
